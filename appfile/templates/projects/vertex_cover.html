Vertex Cover

Quantum Computing in NP problems

This project explores the application of quantum computing techniques to NP-hard problems. The main focus is on leveraging quantum optimization algorithms to find solutions efficiently.

Methodologies Used:

  • Brute-force Implementation.
  • Local QAOA Implementation.
  • Cloud QAOA Implementation.
GitHub Repository →

Project Overview

This project aims to solve the **Vertex Cover problem** using quantum computing. The Vertex Cover problem is a well-known **NP-complete problem** in graph theory, which has applications in network security, bioinformatics, and logistics.

Technologies Used

  • Numpy
  • Networkx
  • Matplotlib
  • Pennylane
  • Dwave-ocean-sdk
  • openqaoa

Challenges & Solutions

  • Brute-Force Solver – Guarantees the best solution but scales poorly.
  • Local QAOA Simulator – Provides approximate solutions but struggles with larger graphs.
  • Cloud-Based QAOA (D-Wave Leap) – A scalable quantum solution with impressive speed.

Key Insights

  • Brute-Force vs. QAOA: Brute-force found the exact minimum vertex cover, while QAOA produced a slightly larger but computationally efficient solution.
  • Scalability: Brute-force failed for large graphs (took 7+ hours), whereas Cloud QAOA solved 10 large graphs (50 nodes each) in just 15 minutes!
  • Quantum in the Cloud: While quantum computing is still evolving, cloud-based quantum optimization proves to be a game-changer for tackling NP-hard problems.
