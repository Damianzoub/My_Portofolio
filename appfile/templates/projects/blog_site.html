Project Overview

Flask Blog is a simple yet powerful blog application built using Flask, featuring essential functionalities such as user authentication, post management, and database integration. This project is designed to demonstrate how to build a web application with Flask while implementing security best practices and database management using SQLAlchemy.

Key Features

  • User Authentication- Secure user login and registration system
  • Post Management- Users can create, edit and delete their blog posts
  • Database Integration- Stores Blog posts and user information using SQLAlchemy
  • Secure Enviroment Variables- Protects sensitive data with .env variables
  • bootstrap for UI- Provides a simple and clean front-end design

Technologies Used

  • FlaskA Lightweight web framework for Python
  • Flask-WTFSecure form handling
  • Flask-LoginUser authentication and session management
  • SQLAlchemyORM for database management
  • SQLite- Lightweight database for storing users and posts
  • bootstrap- Responsive front-end design

Challenges & Solutions

  • User Authentication & Security- Implemented Flask-Login and secure password hashing
  • Database Management- Used SQLAlchemy ORM to simplify interactions with the database
  • Form Validation- Used Flask-WTF for handling user input safely
